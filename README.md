# Streamlit Animal Agriculture App 🌍
 
Over the past 50 years, global meat production has more than tripled, with over 350 million tonnes produced each year. This has drastic effects on the planet such as greenhouse-gas emissions, freshwater use and the expansion of agricultural land. 

This Streamlit application makes predictions on the Earth's future CO² emissions in the next 50 years from animal agriculture.

## 🚀 The Model
Using a **Random Forest** model and datasets provided by the FAO and Our World in Data, this model is trained using extensive data from 173 countries around the world. The data includes: the total population, the number of land animals slaughtered for meat, the resources used for animal agriculture (Land and Water), as well as the greenhouse gases emitted.

## 📊 The Data
The data collected for this model was sourced from:
1. https://www.fao.org/faostat/en/#data
2. https://ourworldindata.org/water-use-stress
3. https://ourworldindata.org/land-use
4. https://ourworldindata.org/population-growth
